ex_camera_set_scroll_offset

Sets the camera's scroll offset

Syntax

ex_camera_set_scroll_offset(cameraHandle, offsetX, offsetY)
Argument Description
cameraHandle The camera handle
offsetX The camera's X offset
offsetY The camera's Y offset
Returns nothing

Description

This function sets the camera's scroll offset values instantly. For example set to 0, 0 in order to reset the offset.

Example

//Set the camera's offset to 100 pixels to the right horizontally and 0 vertically
ex_camera_set_scroll_offset(global._camera, 100, 0);

Minimum Supported Version

exCamera 0.6.0
