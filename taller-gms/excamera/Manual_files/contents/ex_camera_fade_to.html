ex_camera_fade_to

Fades the camera towards a value

Syntax

ex_camera_fade_to(cameraHandle, alpha, duration)
Argument Description
cameraHandle The camera handle
alpha The target alpha value, 0 to 1
duration The duration of the effect, in steps
Returns nothing

Description

This function will fade the camera in or out towards a set value.

Example

//fade the camera towards the value 0.3, the effect lasting 40 steps
ex_camera_fade_to(global._camera, 0.3, 40);

Minimum Supported Version

exCamera 0.5.0
