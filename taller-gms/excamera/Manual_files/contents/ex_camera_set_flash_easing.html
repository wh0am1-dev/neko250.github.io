ex_camera_set_flash_easing

Sets the easing for the flash effect

Syntax

ex_camera_set_flash_easing(cameraHandle, easing)
Argument Description
cameraHandle The camera handle
easing The easing script, pass it without arguments and brackets
Returns nothing

Description

By default the camera uses a smoothstep animation. This function gives more control over this and sets the easing algorithm for all flash effect functions in order to achieve a smoother animation or other effects. You are required to import the supplied scripts first before using an easing function. You may also import only the ones you plan to use.

Example

//Set the easing method for the following flash effect
ex_camera_set_flash_easing(global._camera, ease_out_elastic);

Minimum Supported Version

exCamera 0.8.0
